# **NestNagar UI/UX Design Requirement Document** Version 1.0 Product: NestNagar – Rent & PG Finder Platform: Web App (Mobile First) --- # **1\. Product Goal** NestNagar UI must help users: • Find rooms quickly • Trust listings • Contact owners fast • Navigate easily • Feel safe using platform Primary UX Goal: **User should be able to find a property and start chat within 60 seconds.** Design must prioritize: Speed Clarity Trust Simplicity --- # **2 Design Style Direction** Design Inspiration: Airbnb NoBroker MagicBricks Zillow Design must feel: Modern Clean Trustworthy Housing focused Not overly technical Avoid: Dark tech UI Neon colors Complex dashboards Crypto style design --- # **3 Color System (Final)** ### **Brand Colors** Primary → **\#1B211A** (Headers, navbar, primary actions) Secondary → **\#628141** (Secondary buttons, highlights) Accent → **\#8BAE66** (Active states, selections) CTA Color → **\#E07A2F** (Main action buttons like Chat & Post Listing) Warning → **\#EBD5AB** Error → **\#C94B4B** Background → **\#F7F8F3** White → **\#FFFFFF** --- # **Text Colors** Primary text → **\#1B211A** Secondary text → **\#4B5A48** Muted text → **\#7A8A77** Disabled text → **\#A8B3A5** --- # **Border Colors** Light border → **\#E4E8DE** Default border → **\#D5DDC8** Dark border → **\#B8C5A8** --- # **Status Colors** Success → **\#6FAE4A** Warning → **\#D6A84A** Error → **\#C94B4B** Info → **\#628141** --- # **4 Typography** Fonts: Headings: Inter / Poppins Body: Inter / Roboto Font hierarchy: H1 → 32px → Page title H2 → 24px → Section title H3 → 18px → Card title Body → 16px Small text → 14px Caption → 12px Font weights: Regular 400 Medium 500 SemiBold 600 Bold 700 --- #
